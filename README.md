## 🛠️ Tecnologías Utilizadas
Para la creación de esta web se han separado los lenguajes para mantener un código limpio y profesional:
* **HTML5**: Estructura semántica de las páginas.
* **CSS3**: Diseño vibrante con gradientes y efectos modernos.
* **Markdown**: Documentación del proyecto.

## 📄 Estructura del Proyecto
El sitio se divide en tres secciones principales:
1.  **Inicio**: Presentación de la marca Peps.
2.  **Sobre Nosotros**: Información sobre el autor y la visión del proyecto.
3.  **Copyright**: Registro legal de propiedad intelectual.
